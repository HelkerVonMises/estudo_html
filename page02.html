Mais tags de HTML

- img coloca imagem na page

Uma imagem da Ononoki como exemplo

Dentro de img tem o parâmetro src que define a origem da imagem. Esse parâmetro pode receber um link da imagem ou o endereço da imagem dentro de uma pasta no computador

- alt é uma descrição alternativa para a imagem

- width controla o tamanho da imagem em pixels ou em porcentagem
