Primeiro código em HTML

Começando com as tags


- html e /html definem o início e o fim do código
- head e /head definem o cabeçalho da página e o título
- body e /body definem o corpo da página
- p e /p definem os parágrafos da página
- br é a quebra de linha
- hr coloca uma linha horizontal
- a define um link para ser redirecionado. para ser redirecionado para uma outra página é preciso usar o target como blank
- lorem coloca um texto genérico na tela
